Install two pieces.

The global package contains the local CLI bridge and the built Chrome extension. You install the package once, then load its extension directory into Chromium.

01

Requirements

Before you begin

  • Node.js 18 or newer
  • npm and Git available in your terminal
  • Chrome, Brave, or another Chromium browser with side-panel support
  • An authenticated AI-provider tab
02

Install package

CLI and extension together

Install directly from the project repository:

npm install --global git+https://github.com/RaphaelBlaster/TETHER.git

The repository does not publish a Chrome Web Store package yet. The supported path is Load unpacked.

03

Load extension

Ask TETHER for the exact folder

tether extension-path
  1. Open chrome://extensions.
  2. Enable Developer mode.
  3. Choose Load unpacked.
  4. Select the printed extension\dist directory containing manifest.json.
04

Launch

Start inside your project

tether

Or provide the working project explicitly:

tether -C "C:\path\to\your\project"

Open the side panel beside your authenticated AI chat and activate that tab as the CLI endpoint.

05

Develop from source

Build TETHER itself

git clone https://github.com/RaphaelBlaster/TETHER.git
cd TETHER
npm install
npm --prefix extension install
npm run extension:build
npm install --global .

After rebuilding the extension, reload TETHER in chrome://extensions and refresh the provider tab.
