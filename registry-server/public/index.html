The CLI carries the local bridge. The Chrome extension owns the browser tab. Both come from the same repository.

Read the complete setup guide ↗
  1. 01

    Install CLI + bundled extension

    npm install --global git+https://github.com/RaphaelBlaster/TETHER.git
  2. 02

    Find the unpacked extension

    tether extension-path

    Open chrome://extensions, enable Developer mode, then load the printed dist directory.

  3. 03

    Launch from your working project

    tether

    Open the extension beside an authenticated provider tab and activate it as the CLI endpoint.
